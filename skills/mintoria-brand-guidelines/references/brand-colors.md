# Mintoria Brand Colors

The Mintoria brand uses a curated palette designed to evoke trust, innovation, and stability.

## Core Brand Colors

| Color            | Hex       | Variable               | Usage                                     |
| ---------------- | --------- | ---------------------- | ----------------------------------------- |
| **Brand Blue**   | `#007bff` | `--color-brand-blue`   | Trust, Primary Actions, FinTech stability |
| **Brand Orange** | `#ff6b35` | `--color-brand-orange` | Warmth, Primary CTA, energy               |
| **Brand Green**  | `#28a745` | `--color-brand-green`  | Success, Stability, Growth                |
| **Dark**         | `#121212` | `--color-brand-dark`   | Backgrounds, Headings (dark mode)         |
| **Light**        | `#ffffff` | `--color-brand-light`  | Primary backgrounds, Card surfaces        |

## Functional Palettes

### Primary (Blue)

Use for primary brand identity elements.

- `primary-50`: `#e3f2fd`
- `primary-500`: `#2196f3` (Base)
- `primary-900`: `#0d47a1`

### Accent (Orange)

Use for calls to action and highlighting important information.

- `accent-500`: `#e87f3c` (Main Brand Orange)
- `accent-600`: `#ea580c`

### Success (Green)

Use for optimistic states, completed transactions, and stablecoin safety.

- `success`: `#5fb04c`

## Gradients

Mintoria uses subtle, light gradients to create depth without overwhelming the UI.

- **Primary Gradient**: `linear-gradient(135deg, #2196f3 0%, #1976d2 100%)`
- **Accent Gradient**: `linear-gradient(135deg, #ff6b35 0%, #e65100 100%)`
- **Glow Gradient**: `linear-gradient(135deg, rgba(255, 107, 53, 0.15) 0%, rgba(33, 150, 243, 0.15) 100%)`

# Mintoria Typography

Mintoria uses modern, clean typography to convey professionalism and technological advancement.

## Primary Fonts

- **IBM Plex Sans**: The primary brand font. Used for headings and brand elements.
- **Inter**: Secondary font, used for high-readability body text and UI components.
- **System fallback**: `system-ui, sans-serif`

## Font Weights

- **Logo**: `font-black` (900)
- **Headings (H1, H2)**: `font-bold` (700) or `font-semibold` (600)
- **Body Text**: `font-normal` (400)
- **Buttons / UI**: `font-medium` (500)

## Typical Styles

- **Hero Title**: `text-5xl md:text-7xl font-bold tracking-tight`
- **Section Heading**: `text-3xl font-bold tracking-tight`
- **Body**: `text-lg leading-relaxed text-secondary`

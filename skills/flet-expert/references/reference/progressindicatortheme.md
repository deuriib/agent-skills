# ProgressIndicatorTheme
URL: https://flet.dev/docs/types/progressindicatortheme

ReferenceTypesClassesThemeProgressIndicatorTheme
ProgressIndicatorTheme

Customizes the appearance of progress indicators (ProgressBar, ProgressRing) across the app.

Properties
border_radius - Overrides the default value for ProgressBar.border_radius.
circular_track_color - Overrides the default value for ProgressRing.bgcolor.
circular_track_padding - Overrides the default value for ProgressRing.padding.
color - Overrides the default values for ProgressBar.color and ProgressRing.color.
linear_min_height - Overrides the default value for ProgressBar.bar_height.
linear_track_color - Overrides the default value for ProgressBar.bgcolor.
refresh_bgcolor - Background color of that fills the circle under the RefreshIndicator (TBD).
size_constraints - Overrides the default value for ProgressRing.size_constraints.
stop_indicator_color - Overrides the default value for ProgressBar.stop_indicator_color.
stop_indicator_radius - Overrides the default value for ProgressBar.stop_indicator_radius.
stroke_align - Overrides the default value for ProgressRing.stroke_align.
stroke_cap - Overrides the default value for ProgressRing.stroke_cap.
stroke_width - Overrides the default value for ProgressRing.stroke_width.
track_gap - Overrides the default values for ProgressBar.track_gap and ProgressRing.track_gap.
year_2023 - Overrides the default values for ProgressBar.year_2023 and ProgressRing.year_2023.

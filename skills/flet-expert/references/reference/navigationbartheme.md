# NavigationBarTheme
URL: https://flet.dev/docs/types/navigationbartheme

ReferenceTypesClassesThemeNavigationBarTheme
NavigationBarTheme

Customizes the appearance of NavigationBar controls across the app.

Properties
bgcolor - Overrides the default value for NavigationBar.bgcolor.
elevation - Overrides the default value for NavigationBar.elevation.
height - Overrides the default value for NavigationBar height.
indicator_color - Overrides the default value for NavigationBar.indicator_color.
indicator_shape - Overrides the default value for NavigationBar.indicator_shape.
label_behavior - Overrides the default value for NavigationBar.label_behavior.
label_padding - Overrides the default value for NavigationBar.label_padding.
label_text_style - The style to merge with the default text style for NavigationBarDestination labels.
overlay_color - Overrides the default value for NavigationBar.overlay_color.
shadow_color - Overrides the default value for NavigationBar.shadow_color.

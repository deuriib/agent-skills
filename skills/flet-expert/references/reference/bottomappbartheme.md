# BottomAppBarTheme
URL: https://flet.dev/docs/types/bottomappbartheme

ReferenceTypesClassesThemeBottomAppBarTheme
BottomAppBarTheme

Customizes the appearance of BottomAppBar controls across the app.

Properties
bgcolor - Overrides the default value of BottomAppBar.bgcolor in all descendant BottomAppBar controls.
elevation - Overrides the default value of BottomAppBar.elevation in all descendant BottomAppBar controls.
height - Overrides the default value of BottomAppBar.height in all descendant BottomAppBar controls.
padding - Overrides the default value of BottomAppBar.padding in all descendant BottomAppBar controls.
shadow_color - Overrides the default value of BottomAppBar.shadow_color in all descendant BottomAppBar controls.
shape - Overrides the default value of BottomAppBar.shape in all descendant BottomAppBar controls.

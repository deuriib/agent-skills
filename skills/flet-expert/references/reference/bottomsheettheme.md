# BottomSheetTheme
URL: https://flet.dev/docs/types/bottomsheettheme

ReferenceTypesClassesThemeBottomSheetTheme
BottomSheetTheme

Customizes the appearance of BottomSheet across the app.

Properties
barrier_color - Overrides the default value of BottomSheet.barrier_color in all descendant BottomSheet controls.
bgcolor - Overrides the default value of BottomSheet.bgcolor in all descendant BottomSheet controls.
clip_behavior - Overrides the default value of BottomSheet.clip_behavior in all descendant BottomSheet controls.
drag_handle_color - Overrides the default value of drag handle color in all descendant BottomSheet controls.
elevation - Overrides the default value of BottomSheet.elevation in all descendant BottomSheet controls.
shadow_color - Overrides the default value of shadow color in all descendant BottomSheet controls.
shape - Overrides the default value of BottomSheet.shape in all descendant BottomSheet controls.
show_drag_handle - Overrides the default value of BottomSheet.show_drag_handle in all descendant BottomSheet controls.
size_constraints - Overrides the default value of BottomSheet.size_constraints in all descendant BottomSheet controls.

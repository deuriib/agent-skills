# NotchShape
URL: https://flet.dev/docs/types/notchshape/

ReferenceTypesClassesNotchShape
NotchShape

A shape with a notch in its outline.

Typically used as the outline of a 'host' control to make a notch that accommodates a 'guest' control. e.g the BottomAppBar may have a notch to accommodate the FloatingActionButton.

This class is not intended to be used directly. See usable derivatives:

AutomaticNotchShape
CircularRectangleNotchShape

# Reference Overview
URL: https://flet.dev/docs/reference/

Reference
API Reference
Controls - UI building blocks with properties, events, and usage examples.
Services - Device and platform capabilities such as sensors, storage, and permissions.
CLI - flet commands for creating, running, packaging, and debugging apps.
Types - Core types, enums, events, exceptions, and utilities shared across the SDK.
Built-in binary Python packages for Android and iOS - Prebuilt wheels available from https://pypi.flet.dev for mobile targets.
Environment Variables - Runtime configuration toggles for apps and the CLI.

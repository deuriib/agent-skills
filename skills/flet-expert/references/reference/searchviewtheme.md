# SearchViewTheme
URL: https://flet.dev/docs/types/searchviewtheme

ReferenceTypesClassesThemeSearchViewTheme
SearchViewTheme

Customizes the appearance of SearchBar controls across the app.

Properties
bar_padding - Overrides the default value of SearchBar.view_bar_padding in all descendant SearchBar controls.
bgcolor - Overrides the default value of SearchBar.view_bgcolor in all descendant SearchBar controls.
border_side - Overrides the default value of SearchBar.view_side in all descendant SearchBar controls.
divider_color - Overrides the default value of SearchBar.divider_color in all descendant SearchBar controls.
elevation - Overrides the default value of SearchBar.view_elevation in all descendant SearchBar controls.
header_height - Overrides the default value of SearchBar.view_header_height in all descendant SearchBar controls.
header_hint_text_style - Overrides the default value of SearchBar.view_hint_text_style in all descendant SearchBar controls.
header_text_style - Overrides the default value of SearchBar.view_header_text_style in all descendant SearchBar controls.
padding - Overrides the default value of SearchBar.view_padding in all descendant SearchBar controls.
shape - Overrides the default value of SearchBar.view_shape in all descendant SearchBar controls.
shrink_wrap - Overrides the default value of SearchBar.shrink_wrap in all descendant SearchBar controls.
size_constraints - Overrides the default value of SearchBar.view_size_constraints in all descendant SearchBar controls.

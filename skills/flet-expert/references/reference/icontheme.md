# IconTheme
URL: https://flet.dev/docs/types/icontheme

ReferenceTypesClassesThemeIconTheme
IconTheme

Customizes the appearance of Icon controls across the app.

Properties
apply_text_scaling - Overrides the default value for Icon.apply_text_scaling.
color - Overrides the default value for Icon.color.
fill - Overrides the default value for Icon.fill.
grade - Overrides the default value for Icon.grade.
opacity - An opacity to apply to both explicit and default icon colors.
optical_size - Overrides the default value for Icon.optical_size.
shadows - Overrides the default value for Icon.shadows.
size - Overrides the default value for Icon.size.
weight - Overrides the default value for Icon.weight.

# RadioTheme
URL: https://flet.dev/docs/types/radiotheme

ReferenceTypesClassesThemeRadioTheme
RadioTheme

Defines default property values for descendant Radio controls.

Properties
fill_color - Overrides the default value of Radio.fill_color in all descendant Radio controls.
mouse_cursor - Overrides the default value of Radio.mouse_cursor in all descendant Radio controls.
overlay_color - Overrides the default value of Radio.overlay_color in all descendant Radio controls.
splash_radius - Overrides the default value of Radio.splash_radius in all descendant Radio controls.
visual_density - Overrides the default value of Radio.visual_density in all descendant Radio controls.

# MarkdownStyleSheet
URL: https://flet.dev/docs/types/markdownstylesheet

ReferenceTypesClassesMarkdownStyleSheet
MarkdownStyleSheet

Style overrides for markdown element rendering.

Each field customizes one element group (headings, paragraphs, tables, lists, block quotes, code blocks, etc.). Any None field keeps the control's default styling for that property.

Properties
a_text_style - Text style for links (<a>).
block_spacing - Vertical spacing between markdown block elements.
blockquote_alignment - Alignment for blockquote block content.
blockquote_decoration - Decoration applied to blockquote container.
blockquote_padding - Inner padding for blockquote container.
blockquote_text_style - Text style for blockquote content.
checkbox_text_style - Text style for task-list checkbox labels.
code_text_style - Text style for inline code spans.
codeblock_alignment - Alignment for code block content.

codeblock_padding - Inner padding for fenced code blocks.

"Getting startedInstallation\nInstallation\nPrerequisites​\nPython version​\n\nFlet requires Python 3.10 or later. (1)\n\nCheck your Python version using python --version.\nOperating System​\nmacOS​\n\nFlet supports macOS 12 (Monterey) or later.\n\nWindows​\n\nFlet supports 64-bit version of Microsoft Windows 10 and Windows 11.\n\nLinux​\n\nFlet supports Debian 10, 11 and 12 and Ubuntu 20.04, 22.04 and 24.04 LTS.\n\nDesktop flavor (audio & video support)\nWindows Subsystem for Linux (WSL)\nCreating a virtual environment (venv)​\n\nWe recommend using a virtual environment for your Flet projects to keep dependencies isolated and avoid conflicts with your other Python projects.\n\nFirst, create a new directory for your Flet project and switch into it:\n\nmkdir my-app\n\ncd my-app\n\n\nNext, create and activate a virtual environment (we recommend using uv as package manager):\n\nuv\npip\n\nuv is \"An extremely fast Python package and project manager, written in Rust\".\n\nInstall uv if you haven't already, then run the following commands:\n\nuv init --python='>=3.10'\n\nuv venv\n\nsource .venv/bin/activate\n\n\nInstall Flet​\n\nTo install Flet and add it to your project dependencies, do the following depending on your package manager:\n\nuv\npip\nuv add 'flet[all]'\n\nVerify installation​\n\nTo make sure Flet has been installed correctly, we can check its version using the --version (or -V) flag or the doctor command:\n\nuv\npip\nuv run flet --version\n\n# or\n\nuv run flet doctor\n\n\nNow you are ready to create your first Flet app.\n\nUpgrade Flet​\n\nTo upgrade Flet to its latest version:\n\nuv\npip\nuv add 'flet[all]' --upgrade\n\nEdit this page"

"Getting startedCreating a new Flet app\nCreating a new Flet app\n\nCreate a new directory (or directory with pyproject.toml already exists if initialized with a project manager) and switch into it.\n\nTo create a new \"minimal\" Flet app run the following command:\n\nuv\npip\nuv run flet create\n\nIMPORTANT\n\nAny existing README.md or pyproject.toml (for example, created by uv init) will be replaced by the one created by flet create command.\n\nThe command will create the following directory structure:\n\nREADME.md\n\npyproject.toml\n\nsrc\n\n    assets\n\n        icon.png\n\n    main.py\n\n\nstorage\n\n    data\n\n    temp\n\n\nYou can find more information about flet create command here.\n\nAuto-update​\n\nFlet automatically calls page.update() (or .update() on the nearest isolated ancestor) at the end of every event handler and main() function. This means you don't need to call .update() yourself in most cases:\n\nimport flet as ft\n\n\n\ndef main(page: ft.Page):\n\n    def button_click(e):\n\n        page.controls.append(ft.Text(\"Clicked!\"))\n\n        # no need to call page.update() — it happens automatically\n\n\n\n    page.controls.append(ft.Button(\"Click me\", on_click=button_click))\n\n    # no need to call page.update() here either\n\n\n\nft.run(main)\n\nNOTE\n\nIf your event handler already calls .update() explicitly (e.g. code written for Flet 0.x), the automatic update is skipped to avoid a redundant double update.\n\nDisabling auto-update​\n\nYou can disable auto-update for fine-grained control over when updates are sent to the client. Use ft.context.disable_auto_update() and ft.context.enable_auto_update() to toggle the behavior.\n\nWhen called inside a handler, the setting applies to the current handler context only:\n\nimport flet as ft\n\n\n\ndef main(page: ft.Page):\n\n    def add_many_items(e):\n\n        ft.context.disable_auto_update()\n\n        for i in range(100):\n\n            page.controls.append(ft.Text(f\"Item {i}\"))\n\n        page.update()  # single update for all 100 items\n\n\n\n    page.controls.append(ft.Button(\"Add items\", on_click=add_many_items))\n\n\n\nft.run(main)\n\n\nWhen called outside of event handlers (e.g. at the module level), it controls the global default for the entire app:\n\nimport flet as ft\n\n\n\n# disable auto-update globally\n\nft.context.disable_auto_update()\n\n\n\ndef main(page: ft.Page):\n\n    def button_click(e):\n\n        page.controls.append(ft.Text(\"Clicked!\"))\n\n        page.update()  # must call explicitly since auto-update is off\n\n\n\n    page.controls.append(ft.Button(\"Click me\", on_click=button_click))\n\n    page.update()\n\n\n\nft.run(main)\n\n\nNow let's see Flet in action by running the app!\n\nEdit this page"
